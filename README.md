# GBELS Yar Muhammad Mangrio Digital Hub

A modern, professional school portal built with React, Tailwind CSS, and Framer Motion.

## 🚀 Deployment to Vercel

This project is optimized for Vercel. To deploy:

1. **Push to GitHub**: Upload your code to a GitHub repository.
2. **Connect to Vercel**: 
   - Go to [vercel.com](https://vercel.com).
   - Click "New Project" and select your GitHub repo.
   - The build settings will be automatically detected (Vite).
3. **Deploy**: Click "Deploy" and your site will be live!

## 🛠 Features

- **Trilingual Support**: Sindhi, Urdu, and English.
- **Smart Persistence**: Remembers your page, theme, and language even after refresh.
- **Tools Portal**: School Leaving Certificate generator and more.
- **Digital Learning Hub**: Access to lectures, books, and labs.
- **Premium Design**: Fully responsive with glassmorphism and animations.
- **Live Visitor Counter**: Real-time traffic simulation.

## 📁 Project Structure

- `/src/components`: Reusable UI components.
- `/public`: Static assets (Logos, Banners).
- `vercel.json`: Deployment configuration.
